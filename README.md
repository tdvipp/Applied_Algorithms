# Thuat_toan_ung_dung
Các bài tập trên nền tảng daotao.ai của môn Thuật toán ứng dụng 

## Topics covered

### 1) Graph algorithms / Graph theory (`Mo_phong_ly_thuyet_do_thi`, `Thuc_hanh_buoi_5`)
From files:
- **Minimum Spanning Tree (MST)**
  - Prim (`prim_al.cpp`)
  - Kruskal + **Union-Find / DSU** (`krushkal_union_find.cpp`)
- **Topological Sort** (`topo_sort.cpp`)
- **DFS numbering / low-link arrays** (used for bridge/articulation type problems) (`build_num_and_low_array.cpp`)
- **Bridges & Articulation Points** (`BRIDGES_ARTI_POINTS.cpp`)
- **Strongly Connected Components (SCC)** (`STRONGLY_CONNECTED_COMPONENT.cpp`)
- (Also a graph-based problem) inter-city bus routing (`BUS_INTER_CITY.cpp`)

### 2) Range queries / sparse table-style preprocessing (`Thuc_hanh_buoi_1`)
- **Range Minimum Query (RMQ)** (`range_minimum_query.cpp`)

### 3) Grid / pathfinding / traversal (`Thuc_hanh_buoi_1`)
- **Maze solving** (graph traversal on grid) (`maze.cpp`)

### 4) Stack / histogram / maximal rectangle (`Thuc_hanh_buoi_1`)
- **Largest black sub-rectangle** (commonly solved via histogram + stack) (`largest_black_subrectangle.cpp`)

### 5) Backtracking / combinatorial search (`Thuc_hanh_buoi_2`)
From files:
- `CBUS.cpp` (often appears as a **backtracking / TSP-like** constraint/search problem)
- `BCA.cpp` (problem-specific; likely combinatorial/optimization)

### 6) Counting solutions / integer linear equations (`Thuc_hanh_buoi_2`)
- **Count integer solutions** to a linear equation (`COUNT_INTEGER_LINEAR_EQUATION.cpp`)

### 7) Sorting / divide-and-conquer / BIT-style counting (`Thuc_hanh_buoi_3`)
- **Inversion counting** (`INVERSION.cpp`)

### 8) Greedy / subsequences / searching (`Thuc_hanh_buoi_3`, `De_thi_cuoi_ky/De_1`)
- **Max distance subsequence** (`MAX_DISTANCE_SUB_SEQ.cpp`)
- **LIS (Longest Increasing Subsequence)** (`lis1.cpp`)

### 9) Prefix sums / cumulative sums (`Thuc_hanh_buoi_4`)
- **Max even (subarray) using cumulative sum idea** (`MAX_EVEN_cusum.cpp`)

### 10) Scheduling / optimization (`Thuc_hanh_buoi_5`)
- **Makespan scheduling** (`MAKE_SPAN_SCHEDULE.cpp`)

### 11) Exam / practice sets (`De_thi_giua_ky`, `De_thi_cuoi_ky`)
- Midterm and final exam folders contain **PDFs + solutions** (e.g., `ttud-28-11-2020.pdf`, `CAP (finaltest).pdf`) and problem solutions like `GROUPUP.cpp`, `DIGITS.cpp`, etc.
